# MathMind (offline MVP)

MathMind — это мини-тренажёр на Go, который показывает ход рассуждений при решении задач по математике.

## Что входит
- Один HTTP-сервер на Go (только стандартная библиотека).
- Страница ученика с решением задачи про скидку.
- Подсказки и типы ошибок для неверных веток решения.
- Страница учителя с простой сводкой ошибок в памяти сервера.

## Запуск
```bash
go run .
```

Откройте:
- Ученик: http://localhost:8080
- Учитель: http://localhost:8080/teacher

## Ограничения
- Данные хранятся в памяти процесса (без БД).
- Нет зависимостей, Docker и npm не используются.
